Overview

The ServiceService module creates and manages services and groups them into service collections.

Methods

Create(name, funcTable, propTable, signals, parent): Creates an Service.
GetCreatedService(name): Gets the created service.
GetServices(): Returns all services in the group

Usage Example

local ServiceService = require(game.ReplicatedStorage.ServiceService)

local myServiceGroup = ServiceService.new("MyServiceGroup")

-- Create and add services
local service1 = { Name = "Service1", Type = "Data" }
local service2 = { Name = "Service2", Type = "Network" }

myServiceGroup:AddService(service1)
myServiceGroup:AddService(service2)

print(myServiceGroup:GetServices())

Properties

Name: string - The name of the service group
Services: table - Collection of services in this group

Attributes

CreateActorByService: boolean - Creates an "Actor" class when true.
SetServiceType: dropdown - Sets the type of the service. Can be loaded by MetaLoad.
